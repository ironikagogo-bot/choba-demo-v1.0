# 帳場（ちょうば）— クラウド配信版

営業LINEの負担を減らす秘書サービスのデモ／実行版。ダミー顧客が入った状態で起動し、
「アナウンス（一斉下書き）」「顧客カード」「受信箱」をそのまま触れる。

- 送信機能は持たない（下書きを作るだけ。送信は本人の指）。
- このクラウド版は**ダミーデータ専用**（`CHOUBA_DEMO=1`）。実顧客データは載せない設計。
- AI下書きを本物で動かすには、環境変数 `ANTHROPIC_API_KEY` を設定（未設定でもテンプレ下書きで動く）。

## 公開のしかた（Render）
同梱の `docs/DEPLOY.md` に、GitHubアップロード → Render公開 → APIキー設定までの手順を書いています。

## 構成
- `Dockerfile` … Renderが自動検出してビルド。`uvicorn` で起動、`$PORT` を尊重。
- `requirements.txt` … 依存パッケージ。
- `app/` … 本体（FastAPI + SQLite + 静的UI）。
